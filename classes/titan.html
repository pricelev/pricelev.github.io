Sentinel Classes

Code of the Protector

The code of the protector provides the titan and nearby allies with an overshield and health when performign a melee. Its melee charge flings the titan into battle to start close range. In its super, the titan can create a bubble around himself and his team to protect themselves. Players cant shoot out of the bubble. Upon leaving the bubble players recieve a damage buff.

Code of the Commander

The code of the commander puts out void explosives on enemies that were hit by abilites. These explosives provide the titan with health regin and melee cooldown. While in super the titan can use its alternate fire to put up a shield in front of its allies. Allies can shoot through the shield to do extra damage.

Code of the Aggressor

The code of the agressor uses its melee charge to get into combat faster. After hitting with a melee they recieve cooldown with the melee. \ On grenade kills the grendage cooldown would decrease. The super allows the user to throw its shield more often. The super cooldown is also decreased by the amount of enemies surrounding the player.

Striker Classes

Code of teh Earthshaker

While using code of the earthshaker the titan recieves and additional grenade charge to use. Their melee charges the player into enemives. If the melee hits a target the titan recieve a portion of grenade energy. While in super when the titan slams the ground it leaves a damage field on the ground.

Code of the Missle

The code of the missle allows the user to use their melee ability while in the air. Their melee slams into the ground and then gives super energy and other buffs. Some of these buffs help with reload and ammo consumption for the titan. The super turns the titan into a flying missle that can be targetable.

Code of the Juggernaut

The code of the juggernaut uses its melee ability to increase weapon damge and reload speed. On melee kills the titan will regin health. While getting kills in supers the titan will increase the duration. This super is highly used in PVE content.

Sunbreaker Classes

Code of the Fire-Forged

The code of the fire-forged is used mainly in PVE as it provides debuffs to enemies. This debuff is spread by using the melee ability on the enemy. While in super the titan throws hammer. In code of fire-forge when killing enemies with this hammer cuases them to explode.

Code of the Devastator

The code of the devastator turns the melee charge of the titan into a throwing hammer that can be picked up to refund the melee charge. Gitting kills with solar abilities increasethe damge of the titans abilites. During the super the titan turns its small hammer into a large hammer that they can spin to cause damage to enemies.

Code of the Siegebreaker

In code of the siegebreaker the titan creates solar flares on ability kills. While standing in these solar flairs the Titan and other players recieve a increase in reload speed. The super is similar to the code of the fire-forge except it creates solar spots at the enemy locations.

Stasis Class

Behemoth

The behemoth subclass is new to the game and allows the titan to use cosmicice to shatter and freeze enemies. The titan recieves a powerful slide that can shatter enemies and is used for movement in PVP. The super slams the ground multiple times and sends out pillars of ice that can be shattered to cause lots of damage to enemies.
